# Cricket Dashboard

A lightweight Python application that displays live cricket scores, upcoming fixtures, and player statistics from ESPN Cricket API in an interactive terminal dashboard with CSV export capabilities.

**Quick Start:**
```bash
pip install requests beautifulsoup4 rich
python cricket.py
```

**Features:** Live scores • Upcoming fixtures • Player stats • Auto-refresh (30s) • CSV export • Multi-league support (IPL, BBL, T20 Blast, and more)
